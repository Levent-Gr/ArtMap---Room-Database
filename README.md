
# ArtMap

The purpose of this project is to record the location of users by photographing artistic objects in the places they visit.
This project was taken from the AtilSamancioglu Udemy course.


## Libraries And Tools

- Used language Java
- View Binding
- Room Database
- RxJava
- RecyclerView
